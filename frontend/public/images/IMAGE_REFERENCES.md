# Image References for WebVibes Technology Portal

This document contains all Unsplash image URLs used throughout the portal. Images are loaded directly from Unsplash CDN with optimized parameters.

## URL Parameters
- `w` - Width in pixels
- `h` - Height in pixels  
- `q` - Quality (1-100)
- `fit=crop` - Crop to exact dimensions
- `auto=format` - Automatically serve WebP when supported

## Usage Guidelines

### Lazy Loading
All images should use Angular's lazy loading directive:
```html
<img [src]="imageUrl" loading="lazy" alt="Description">
```

### Responsive Images
For hero sections, use CSS background images with multiple breakpoints:
```css
.hero {
  background-image: url('image-url?w=768&h=400');
}

@media (min-width: 768px) {
  .hero {
    background-image: url('image-url?w=1920&h=600');
  }
}
```

### Error Handling
Always provide fallback images:
```html
<img [src]="imageUrl" 
     (error)="handleImageError($event)"
     loading="lazy" 
     alt="Description">
```

### Performance Tips
1. Use appropriate image sizes for each viewport
2. Set explicit width and height to prevent layout shift
3. Use `loading="lazy"` for below-the-fold images
4. Implement intersection observer for scroll animations
5. Consider using WebP format with fallbacks
